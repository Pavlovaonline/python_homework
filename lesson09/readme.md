#README

Телеграм бот "Телефонный справочник"
Имя бота в телеграм: GB_Python_hw08

Автор: Павлова Е.П.
https://github.com/Pavlovaonline

Для запуска бота отправьте сообщение /start.

Далее вы можете выбрать необходимое действие:
    Найти контакт: /search  
    Добавить новый контакт: /add

Если вы хотите удалить или изменить уже существующую запись в справочнике, то сначала найдите необходимый контакт по имени (через коменду /search). Далее вы можете выбрать соответствующее действие: 
    Изменить контакт /change
    Удалить контакт /delete

Чтобы изменить данные контакта, после команды /change вы можете выбрать поле, которое необходимо изменить: Name, Number, Category, Comment.
После выбора поля укажите новые данные.

Для удаления контакта из справочника достаточно отправить команду /delete.

Для добавления нового контакта в справочник вам необходимо указать данные для каждого поля (Name, Number, Category, Comment), после чего новый контакт будет сохранён.

Чтобы запустить бот снова отправьте /start.
